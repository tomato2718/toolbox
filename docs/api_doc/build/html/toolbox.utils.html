toolbox.utils.parse_unknown_args(args: list[str], prefix: str = '-') → dict¶

Parse unknown arguments list into dict. Basically, it’s design to parse the return of ArgumentParser.parse_known_args().

For example, a list like:

[
    "test",
    "--foo",
    "bar",
    "-buz",
    "1",
    "2",
    "3",
    "--sys",
    "xyz"
]

Will be parsed into:

{
    "unnamed": "test",
    "foo": "bar",
    "buz": [
        "1",
        "2",
        "3"
    ],
    "sys": "xyz"
}
Parameters:

  • args (list) – List of arguments.

  • prefix (str) – The prefix of arguments.

Returns:

Parsed arguments.

Return type:

dict
